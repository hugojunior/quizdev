Which functions can be used to get the image's properties (size, width, and height)?

  1. The functions are getimagesize() for size, imagesx() for width and imagesy() for height.
  2. The functions are getimagesizeof() for size, imagesxn() for width and imagesyn() for height.
  3. The functions are getsizeimage() for size, imagesx() for width and imagesy() for height.
