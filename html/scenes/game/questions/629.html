Em PHP, quais funções podem ser usadas para obter as propriedades da imagem (tamanho, largura e altura)?

  1. getimagesize() para tamanho, imagesx() para largura e imagesy() para altura.
  2. getimagesizeof() para tamanho, imagesxn() para largura e imagesyn() para altura.
  3. getsizeimage() para tamanho, imagesx() para largura e imagesy() para altura.
Tema: PHP
